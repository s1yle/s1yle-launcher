# 简介

本项目为开源项目，欢迎开发者参与贡献，共同优化启动器功能：

- 功能建议：可在 `github issue` 中创建并提交建议，或联系我反馈；

- 代码贡献：`Fork` 本项目，提交 `Pull Request`，经审核通过后即可合并；

所有贡献者将在 仓库的 `contributors` 和 `启动器“贡献名单” (待开发)` 中展示，感谢每一位参与者的支持！
