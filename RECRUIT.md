# WeCraft! Launcher - 招募志同道合之人

> 一个 **玩家/服主身份一键切换** 的 Minecraft 启动器

## 我们在做什么

WeCraft! Launcher 不是一个普通的启动器。我们的目标是搭建一个 **中小服主与玩家之间的桥梁**：

- **服主** 可以免费上传服务器，一键管理配置、查看数据看板
- **玩家** 可以免费发现社区服务器，一键加入畅玩
- **身份一键切换** — 你既是玩家，也可以是服主

## 技术栈

| 前端 | 后端 | 桌面 |
|------|------|------|
| React 19 + TypeScript | Rust (Tokio) | Tauri 2 |
| Tailwind CSS 4 + Framer Motion | reqwest + serde | 跨平台 |
| Zustand + i18next | 模块化命令架构 | Windows/Linux/macOS |

# WeCraft! Launcher - Join Us

> A Minecraft launcher with **one-click role switching between player and server owner**

## What We're Building

WeCraft! Launcher is not just another launcher. We aim to build a **bridge between small/medium server owners and players**:

- **Server Owners** can upload their servers for free, manage configs, and view analytics dashboards
- **Players** can discover community servers for free and join with one click
- **One-click identity switch** — Be a player and a server owner at the same time

## Tech Stack

| Frontend | Backend | Desktop |
|----------|---------|---------|
| React 19 + TypeScript | Rust (Tokio) | Tauri 2 |
| Tailwind CSS 4 + Framer Motion | reqwest + serde | Cross-platform |
| Zustand + i18next | Modular command architecture | Windows/Linux/macOS |
